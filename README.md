# Prime Finish Mobile Detailing

This project is prepared for simple client hosting as a static website.

## Recommended Hosting

Use Netlify for the easiest handoff:

1. Push this repo to GitHub.
2. In Netlify, choose `Add new site` and import this repository.
3. Keep the publish directory as `public`.
4. Deploy the site.

## Booking Form

The booking form in `public/contact.html` is configured for Netlify Forms.

After deployment, form submissions will appear in the Netlify dashboard under:

- `Forms`

## Local Preview

You can still preview the site locally with:

```bash
npm start
```

Then open:

```text
http://localhost:3000
```
